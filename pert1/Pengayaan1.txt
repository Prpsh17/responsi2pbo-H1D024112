Pertanyaan Refleksi
Setelah menyelesaikan kode, jawab pertanyaan berikut:

Jelaskan perbedaan antara fungsi (function) dan prosedur (procedure) dalam pemrograman berorientasi objek.
Mengapa penting menggunakan konstanta (final variables) dalam suatu kelas? Apa keuntungannya?
Apa yang terjadi jika kita mencoba mengubah nilai suatu konstanta setelah objek dibuat? Berikan contoh.
Bagaimana cara mendefinisikan konstanta dalam bahasa Java?

Jawaban

1. perbedaan antara fungsi dan prosedur adalah 1. fungsi menggunakan return prosedur tidak

2. Konstanta (final) digunakan untuk nilai yang tidak boleh berubah selama program berjalan.
Keuntungannya:
1. Menjaga konsistensi data
Nilai penting (misal faktor aktivitas, tarif tetap, rumus perhitungan) tidak akan berubah secara tidak sengaja.
2. Mencegah error logika
Karena programmer lain tidak bisa mengubahnya sembarangan.
3. Lebih aman dan mudah dipahami
Pembaca kode tahu bahwa nilai tersebut bersifat tetap dan digunakan sebagai standar.

3. Java akan memberikan error saat kompilasi, karena variabel final tidak boleh diubah setelah diberi nilai.

4. Gunakan keyword final, dan biasanya static jika digunakan sebagai konstanta kelas.
